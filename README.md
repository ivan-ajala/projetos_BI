## 🚀 Como Executar o Projeto 1 (ETL Vendas)

Para replicar o pipeline completo:

1. **Clone o repositório:**
```bash
git clone [URL_DO_REPOSITORIO]
cd projeto-1-etl-vendas
```

2. **Execute o pipeline ETL:**
```bash
python src/extract_data.py
python src/transform_data.py  
python src/load_data.py
```

3. **Tableau:**
- Conecte ao `tableau_main.csv` gerado
- Importe o arquivo `.twb` da pasta `/dashboard`
