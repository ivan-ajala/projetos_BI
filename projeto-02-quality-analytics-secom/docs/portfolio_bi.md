
# Projeto para Portfólio de BI / Analytics

## Título sugerido

Manufacturing Quality Analytics – SPC & Failure Monitoring (UCI SECOM)

## Resumo (para CV / LinkedIn)

Desenvolvi um pipeline de dados e um dashboard de qualidade para uma linha de manufatura de semicondutores, integrando leituras de 500+ sensores a KPIs de processo (yield, %FAIL, estabilidade de processo). Apliquei conceitos de SPC (limites de controle, análise de variabilidade), análise estatística e visualizações em Power BI para identificar sensores críticos, monitorar desvios de processo e suportar decisões de melhoria contínua. O projeto simula um ambiente real de Quality Management, com foco em visibilidade executiva, alertas proativos e análise de causa raiz orientada a dados.

## Pontos para destacar em entrevistas (BI)

- KPIs de qualidade: yield, %FAIL, estabilidade do processo.
- Uso de SPC (pseudo control charts) para monitoramento de processo.
- Estruturação de dashboard em abas (Overview, Sensores Críticos, SPC, Root Cause).
- Conexão com rotina de qualidade: monitorar, investigar, agir.
